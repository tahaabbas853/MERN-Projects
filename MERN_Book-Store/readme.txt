Project name: TS Book Store 
Description: Web application with CRUD operations using MERN stack
Future Scope: Will add login/registeration and other functionalities later.
=========================================================================

Steps to run the Project
=============================

step-1:
======
Install node js (if you haven't installed)

step-2:
======
Login in the mongodb database and create a new cluster and database, I have used mongo compass.
For mongo atlas, change the mongodb string in the (.env) file (MONGOURL) with own database, username and password

step-3:
======
Open terminal and go to the client directory and run command: npm run dev
go to the server directory and run command: npm start
